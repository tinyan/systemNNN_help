init - printcg

FileName = init\printcg.xtx

共通設定
背景スクリプト設定 不可
BGM設定 可
その他設定 なし


専用設定
名前 サブ デフォルト値 解説
modename printCG モード名、変更不可。省略不可
scrollDirectionX 0 標準スクロール方向X 0:→ 1:←
scrollDirectionY 0 標準スクロール方向X 0:↓ 1:↑
scrollTimeX 30 横スクロール時間をフレーム数で設定
scrollTimeY 30 縦スクロール時間をフレーム数で設定
scrollWaitX 0 横スクロール開始までの待ち時間をフレーム数で設定
scrollWaitY 0 縦スクロール開始までの待ち時間をフレーム数で設定
backMode selectcg 右クリックで戻る画面を番号またはモード名で設定
nextSound 3 次へ進む操作を行った場合の効果音を番号で設定
prevSound 3 1つ前に戻る操作を行った場合の効果音を番号で設定
backSound 2 CG鑑賞を終了する操作を行った場合の効果音を番号で設定
taBGFlag 0 taを表示するとき背景を 0:塗りつぶし 1:絵を表示 2:キャラごとに絵を表示
taBGColorR 0 塗りつぶし色赤
taBGColorG 0 塗りつぶし色緑
taBGColorB 0 塗りつぶし色青
taBGFileName 共通背景絵
taBGCharaFileName[n] キャラごとの背景絵 
cgVoiceWaitTime 10

CGが切り替わってから音声がなりはじめるまでの時間をフレームで指定
