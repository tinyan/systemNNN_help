ボタン設定
ボタン名 sceneButton この名前でボタンを作成するように記述すること
名前 サブ デフォルト値 解説
name sceneButton 省略しないこと
addPartsNumber 0 飾りの個数(最大3個まで)
kazari[k]printX 0 [k]番目のかざりの表示位置をボタンからの相対座標で設定
kazari[k]printY 0
kazari[k]sizeX 16 [k]番目のかざりの大きさを設定
kazari[k]sizeY 16
kazari[k]fileName [k]番目のかざりのファイル名を設定
