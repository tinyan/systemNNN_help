代入演算子




NNNシステムで使用できる代入演算子は、
通常代入と、一部の2項演算代入子です

代入演算子は、シナリオの「計算」コマンド内でのみ使用できます
= 代入 左辺に右辺の値または計算結果を代入します。
+= 演算代入 左辺に右辺の値を=の左側の演算子で計算し代入します
-=
*=
/=
%= 

代入の使用例

館で落とした人数 = 7;
崖から落とした人数 = 1;
落とした人数 = 館で落とした人数 + 崖から落とした人数;
ミスラのいんと = 2 + 2;


上の計算を実行すると、「落とした人数」は8に、「ミスラのいんと」は、4になります




演算代入の使用例

食べたパンの枚数 = 1;
食べたパンの枚数 += 3;
食べたパンの枚数 += 3;


上の計算を実行すると、「食べたパンの枚数」は7になります
