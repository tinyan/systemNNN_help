エフェクト - 雪





専用雪画像を使用した降雪エフェクトです

【レイヤーエフェクト】
【1個制限】(注)
【専用画像必要】
【2D3D:???】
【永続エフェクト】のはず・・・

雪画像ファイル ta_eff_snow または eff_snowが必要です
パラメーター名デフォルト値解説最小値最大値
フレーム 20 エフェクトの長さをフレーム数で指定 0 9999
近い雪タイプ 0 表示する最も近い雪のタイプを指定 0 5
遠い雪タイプ 5 表示する最も遠い雪のタイプ指定 0 5

雪のタイプは数値が小さいほど近い(大きい)

(注)
近い雪タイプ〜遠い雪タイプが表示されますが
全レイヤーで重複がなければ複数のレイヤーに使用することができます
