エフェクト - スタッフロール



上下で透明になるエフェクトで、スタッフロールに便利です
絵の移動は座標指定で行います


【画像エフェクト】
【2D3D:???】


パラメーター名デフォルト値解説最小値最大値
フレーム 20 エフェクトの長さをフレーム数で指定 0 9999
透過タイプ 2 透過ありのみ実装 1 2
透明度％１ 0 0 100
透明度％２ 50 0 100
透明度％３ 100 0 100
透明度％４ 100 0 100
透明度％５ 50 0 100
透明度％６ 0 0 100
座標２ 0 0 600
座標３ 120 0 600
座標４ 360 0 600
座標５ 480 0 600
枚数 1 分割した絵をまとめて処理する場合にその合計枚数を 1 9999
加占有レイヤ 0 分割処理を行う場合に、使用する下のレイヤー数を指定。通常は2または(枚数-1)を指定する 0 16
画像を縦に分割して、それぞれに画面座標に応じた透明度を
なめらかに設定します
