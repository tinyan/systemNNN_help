init - omake

FileName = init\omake.xtx

共通設定
背景スクリプト設定 可
BGM設定 可
その他設定 メニュー 最初の4つは、CG鑑賞、シーン鑑賞、音楽鑑賞、タイトルに戻る


専用設定
名前 サブ デフォルト値 解説
modename title モード名、変更不可。省略不可
fileNameBG title_bg2 背景画像名
menuStartWaitTime 0 メニューが表示し始めるまでの時間をフレーム数で設定する
menuStartEffectTime 0 メニューが表示されはじめてからの特殊エフェクトがかかっている時間をフレーム数で設定する
menuStartEffectType 0 特殊エフェクトの種類 0:移動 1:半透明
menuStartDeltaX 0 特殊エフェクトで移動を選んだ場合のメニューの表示開始位置を相対座標で設定
menuStartDeltaY 0
autoCGCharaFlag 0 1:CG鑑賞キャラ選択を行わない(1人)
autoSceneCharaFlag 0 シーン鑑賞キャラ選択を行わない(1人)
cgModeExistFlag 1 0:CG鑑賞ボタンを削除する
sceneModeExistFlag 1 0:シーン鑑賞ボタンを削除する
bgmModeExistFlag 1 0:音楽鑑賞ボタンを削除する
追加 miniGame ミニゲームの個数 miniGameCheckVarName 使えるミニゲームの個数を変数で制御する場合その変数名をいれる miniGameCheckX[n] miniGameCheckY[n] miniGameCheckSizeX[n] miniGameCheckSizeY[n] ミニゲームに反応する隠しエリアの座標とサイズを設定する
