init - selectscenechara

FileName = init\selectSceneChara.xtx

共通設定
戻るボタン設定 必要 それぞれのとこを参照
背景スクリプト設定 可能 それぞれのとこを参照
BGM設定 可能 それぞれのとこを参照
画面切り替え時フェード設定 可能 それぞれのとこを参照
画面切り替え時音声設定 可能 それぞれのとこを参照
その他設定 なし それぞれのとこを参照


専用設定
名前 サブ デフォルト値 解説
modename selectSceneChara モード名、変更不可。省略不可
filenameBG scenem_bg 背景画像ファイル名

filenameButton ta_scenem_bt キャラ絵のファイル名
cursorMode 1 0:カーソルなし 1:色 2:絵
cursorPercent 50 カーソルを半透明で表示する場合の透明度
filenameCursor ta_cgm_cur カーソルを絵で表示する場合のファイル名
cursorPrintType 1 カーソル画像の表示方法 0:通常 1:半透明 2:加算
checkOnByPicFlag 0 カーソルオンを 0:大きさ 1:画像で チェックする
picNumberX シーンキャラの人数 シーンキャラ画像が横に何人ずつならべてあるか
numberX シーンキャラ人数 並べて表示するシーンキャラの数を縦横それぞれ指定する
numberY 1
sizeX 127 シーンキャラのサイズ
sizeY 127
baseX 0 シーンキャラ表示開始位置
baseY 0
nextX sizeX 次のシーンキャラまでの距離
nextY 0
heartPrintFlag 1 1:特殊シーンにハートマークなどを追加表示する
heartPrintX 0 ハートマークの表示位置をキャラからの相対座標で設定
heartPrintY 0
heartSizeX 16 ハートマークのサイズ
heartSizeY 16
filenameHeart ta_heart ハートマークのファイル名
combineFlag 0 1:シーンキャラ表示位置を組み合わせで作成する
combineType 0 組み合わせを 0:右上 1:左下 に、並べる
combineNumber 1 組み合わせの人数
customPrintFlag 0 1:シーンキャラ表示位置を個別に設定する
customStart 1 個別設定するキャラの開始番号
customEnd シーンキャラの人数 個別設定するキャラの終了番号
printX[n] [n]番目のシーンキャラの表示位置
printY[n]
cursorColorR 128 カーソルの色
cursorColorG 0
cursorColorB 255
totalPercentPrintFlag 1 1:全シーン回収率を表示する
totalPercentPercentFlag 1 1:%(パーセント)記号も表示する
fileNameTotalPercentPic ta_font_cgm1 シーン回収率表示用の数字フォントグラフィックファイル名
totalPercentPrintX シーン回収率の表示位置
totalPercentPrintY
totalPercentSizeX シーン回収率数字フォントのサイズ
totalPercentSizeY
totalPercentNextX totalPercentSizeX シーン回収率数字フォントの文字間隔
charaPercentPrintFlag 1 1:キャラごとのシーン回収率を表示する
percentPercentFlag 1 1:%(パーセント)記号表示する
fileNamePercentPic ta_font_cgm1 シーン回収率表示用の数字フォントグラフィックファイル名
percentPrintX シーン回収率表示位置をキャラ絵の表示位置からの相対座標で設定する
percentPrintY
percentSizeX シーン回収率数字フォントのサイズ
percentSizeY
percentNextX percentSizeX シーン回収率数字フォントの文字間隔
charaVoiceFlag 0 1:キャラ絵にカーソルがのったときに音声を出す(未実装)

charaVoiceFileName[n] 音声ファイル名
