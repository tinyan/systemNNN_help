共通設定
名前/サブ デフォルト値 解説
enterFadeInType 0 0:フェードイン処理なし。-1:塗りつぶし色からのフェードイン 1〜:ワイプ処理
enterFadeInTime 10 フェードイン時間をフレーム数で設定
enterFadeInColorR 0 フェードイン塗りつぶし処理の場合の色をRGBで設定
enterFadeInColorG 0
enterFadeInColorB 0
createExitScreenFlag 1 0:フェードイン、アウト用の画像を作成しない
fillExitScreenFlag 0 1:終了時に次のフェードインでつかわれる画像を塗りつぶす
exitFadeOutType 0 0:フェードアウト処理なし。1〜:ワイプ処理でフェードアウト
exitFadeOutTime 10 フェードアウト時間をフレーム数で設定
exitFadeOutColorR 0 フェードアウト塗りつぶし色をRGBで設定
exitFadeOutColorG 0
exitFadeOutColorB 0
createExitScreenFlagSpecial 0 1:特殊フェードイン、アウト用の画像を作成する
exitFadeOutSpecialMode -2 ここで設定されたモードに変化する場合、以下の設定が使われる(標準モード名で設定可能)
exitFadeOutTypeSpecial 0 0:特殊フェードアウト設定なし。1〜:ワイプ処理でフェードアウト
exitFadeOutTimeSpecial 10 特殊フェードアウト時間をフレーム数で設定
exitFadeOutColorSpecialR 0
exitFadeOutColorSpecialG 0
exitFadeOutColorSpecialB 0
fillExitScreenFlagSpecial 0 1:終了時に次のフェードインでつかわれる画像を塗りつぶす（特殊フェードアウト用） 

特殊フェードアウト設定は、変化先が複数ある画面モードで(CONFIG等)
変化先に応じてフェードアウトを設定するためのものです

たとえば、CONFIG画面で通常はフェードアウトありにして、
(ゲーム中によばれた場合など、戻り先でフェードイン設定ができないため)

タイトルに戻るときのみフェードアウトなしに設定するとよいようです

デフォルトではConfig Load画面では
特殊フェードアウト設定は
もどりさき:TITLE
