ロゴ



ロゴ画面の製作例です
画像の準備と設定 
表示するロゴの個数を決めて、画像を準備します。
サイズは、ゲーム画面と同じにしてください。
形式はjpgかbmpです

準備した画像は、
bmp/sys
または
jpg/sys
フォルダーにいれてpacnyanで変換しておきましょう。

ここでは、nnnのロゴと、サークルのロゴの2枚を表示することにします。
logo1 logo2 


init設定
ファイル init/logo.xtx

modename logo

logoNumber 2


個別にファイル名を設定する機能もありますが、はじめからlogo1,logo2...
という名前にしてしまうのが簡単です
表示のさせ方のカスタマイズ 
1枚ののロゴは、短く、2枚目のロゴは長めに表示
1枚目は、5フレームでフェードイン、10フレーム維持、3フレームでフェードアウト
2枚目は、30フレームでフェードイン、200フレーム維持、50フレームでフェードアウト
という設定にしてみます

useCustomTimeFlag 1

fadeInTime1 5
keepTime1 10
fadeOutTime1 3

fadeInTime2 30
keepTime2 200
fadeOutTime2 50
フェードイン、フェードアウトの色のかすたまいず 
1枚目は黒からフェードインし、青にフェードアウト
2枚目は青からフェードインし、白にフェードアウト

useCustomColorFlag 1

fadeInColorR1 0
fadeInColorG1 0
fadeInColorB1 0

fadeOutolorR1 0
fadeOutColorG1 0
fadeOutColorB1 255

fadeInColorR2 0
fadeInColorG2 0
fadeInColorB2 255

fadeOutolorR2 255
fadeOutColorG2 255
fadeOutColorB2 255
マウスクリックによるロゴ表示スキップのカスタマイズ 
サークルロゴのみスキップ不可にしてみる

useCustomSkipFlag 1

skipOk2 0


ロゴをスキップ不可にするのは、あまり好まれないようです
契約上の理由等で、どうしようもない場合以外は設定しないことをお勧めします

その他の設定は INIT仕様 - ロゴ を参照してください
