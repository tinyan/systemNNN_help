init - title

FileName = init\title.xtx

共通設定
背景スクリプト設定 可
BGM設定 可
その他設定 メニュー 最初の5つは、ゲーム開始、ロード、環境設定、おまけ、ゲーム終了


専用設定
名前 サブ デフォルト値 解説
modename title モード名、変更不可。省略不可
fileNameBG title_bg 背景画像名
autoRestartTime 0 1〜:指定されたフレーム数経過後、restartNextModeに移行する
scriptEndIsModeEnd 0 1:背景スクリプト使用時、スクリプト終了後restartNextModeに移行する
restartNextMode nothing 使用する場合は、なるべくnothingではなく、ゲームごとに実行されるnothingの次のモードを入れる
menuStartWaitTime 0 メニューが表示し始めるまでの時間をフレーム数で設定する
menuStartEffectTime 0 メニューが表示されはじめてからの特殊エフェクトがかかっている時間をフレーム数で設定する
menuStartEffectType 0 特殊エフェクトの種類 0:移動 1:半透明※
menuStartDeltaX 0 特殊エフェクトで移動を選んだ場合のメニューの表示開始位置を相対座標で設定
menuStartDeltaY 0
