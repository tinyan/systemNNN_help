init - messageCursor

FileName = init\messageCursor.xtx


n:1 通常ウィンドウ用送りカーソル
n:2 通常ウィンドウ用自動カーソル
n:3 全画面用送りカーソル
n:4 全画面用自動カーソル

名前 サブ デフォルト値 解説
name messageCursor 省略しないこと
filenameCursor[n] mes_cur カーソルグラフィックファイル名
fullFixX 0 1:全画面カーソルで表示座標を固定する
fullFixX 0
sizeX[n] 32 カーソルのサイズ
sizeY[n] 32
printX[n] 0 カーソルの相対表示位置
printY[n] 0
animePattern[n] 1 カーソルのアニメパターングラフィックの数
animeSpeed[n] 1 アニメ速度をフレーム数で設定
animeReverse[n] 0 1:アニメパターンを折り返す
percentStart[n] 100 開始透明度
percentEnd[n] 100 終了透明度
percentLoop[n] 1 透明度処理のループ数
percentReverse[n] 0 1:透明度処理で折り返し処理を行う
cursorNumberConfig 1
cursorNumberScript 1


type:送りカーソルの用途別種類(1-4)
m:スクリプトによる送りカーソルの種類
n:コンフィグによる送りカーソルの種類
filenameCursor[type]_[m]_[n] config,scriptともに複数種類ある場合
filenameCursor[type]_[m] scriptのみ複数種類
filenameCursor[type]_[n] configのみ複数種類
