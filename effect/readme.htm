エフェクト - 基本解説

エフェクト種類
【画像エフェクト】同一レイヤーの画像データに対してエフェクト処理を行います
【レイヤーエフェクト】その時点での途中画像に対してエフェクト処理を行います


2D3D:2DExe,3DExe対応
【2D】2DExeのみ実装
【3D】3DExeのみ実装
【2D3D】どちらにも実装
【???】現在未定
現時点でのエフェクトは2D3D共に動作予定ですが、今後追加するエフェクトは
3D専用になる可能性があります
(現在3Dプログラムは未公開です)

【1個制限】:全レイヤーで1個しか使用できません


ムービー作成専用:複雑な処理を行い非常に処理が重いため
実質ムービー作成用にしか使えません



それぞれパラメータ指定により細かな動作が指定可能です
