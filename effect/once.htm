エフェクト - ONCE


最初の1(または数フレーム)のみ画像を表示します




【2D3D:???】

パラメーター名 デフォルト値 解説 最小値 最大値
フレーム 20 エフェクト全体の時間をフレーム数で指定 0 9999
開始フレーム 0 0 9999
長さ 1 0 9999 

ONCEはPUTの補助用エフェクトです
多くの場合PUTでは、1フレーム目に表示すべき画像が
GETバッファーに入っていない場合があります
そのかわりに画像を表示されるためのエフェクトとしてONCEが準備されています

使用例
0:PUT
1:ONCE
2〜12:いろいろ(画像の拡大や移動がよく使われます)
13:GET

これで前の画像に次々と演出をかさねながら動かすことができます
